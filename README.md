# Livestock Biometric Registry

A role-based biometric livestock identification system leveraging deep learning embeddings and cosine similarity matching.

## Features

- Biometric embedding extraction (MobileNetV2)
- Cosine similarity-based verification
- Duplicate detection with override control
- Authority-level global search
- Farmer-specific ownership isolation
- Adjustable verification threshold
- Modular architecture (GUI, Core, Database)

## Architecture

- GUI Layer (Tkinter)
- Core ML Engine (PyTorch)
- JSON-based Registry Storage
- Role-Based Access Control
