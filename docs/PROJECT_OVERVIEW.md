# Project Overview

## 1. Introduction

Livestock theft and misidentification remain significant challenges in agricultural environments. Traditional tagging systems can be removed, forged, or damaged.

This project proposes a biometric livestock identification system using muzzle patterns, which are permanent and unique to each animal.

## 2. Objectives

- Develop a biometric identification system using deep learning embeddings.
- Implement role-based access control.
- Prevent duplicate livestock registration.
- Enable ownership-based livestock verification.

## 3. Scope

This project delivers an MVP (Minimum Viable Product) demonstrating feasibility. It includes:

- Authority-level livestock registration
- Duplicate detection
- Global livestock search
- Farmer-specific verification
- Embedding-based identity matching

## 4. Limitations

- Curated dataset used for demonstration.
- Linear similarity search used for MVP.
- No cloud deployment in current version.
